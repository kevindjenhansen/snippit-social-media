# Snippit

A small social media feed application built for a Noroff front-end assignment. Snippit lets users register, log in, create posts, edit their own posts, delete posts, view single posts, and search the feed.

## Live Site

[View Snippit on GitHub Pages](https://kevindjenhansen.github.io/snippit-social-media/index.html)

## Project Overview

Snippit is a lightweight social media platform focused on core post-feed functionality. The project uses the Noroff API and requires a valid Noroff student account to authenticate and interact with posts.

The goal of the project was to build a clean, responsive front-end application using HTML, JavaScript, Bootstrap, and Sass, while working with API data, authentication, and CRUD functionality.

## Features

- User registration with `@stud.noroff.no` email
- User login and logout
- Create new posts
- Edit existing posts
- Delete own posts
- View a single post page
- Search posts by title
- Profile page showing user posts
- Responsive layout
- Sass-based styling
- Bootstrap included through Sass

## Built With

- HTML
- JavaScript
- Bootstrap
- Sass / SCSS
- Noroff API

## Getting Started

### Prerequisites

You need the following installed:

- [Node.js](https://nodejs.org/)
- npm
- VS Code or another code editor
- Live Server extension, or another local server

You also need:

- A valid `@stud.noroff.no` email
- Access to the Noroff API
- A Noroff API key

API key documentation:  
[https://docs.noroff.dev/docs/v2/auth/api-key](https://docs.noroff.dev/docs/v2/auth/api-key)
